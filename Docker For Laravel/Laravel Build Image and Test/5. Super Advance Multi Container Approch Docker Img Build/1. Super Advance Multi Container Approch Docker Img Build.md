## 2. Multi-Container Dockerfiles

### docker/php/Dockerfile (Laravel Application Container)
```dockerfile
# ==================================
# STAGE 1: Composer Dependencies
# ==================================
FROM composer:2 AS vendor

WORKDIR /app
COPY composer.json composer.lock ./

RUN composer install \
    --no-dev \
    --no-interaction \
    --prefer-dist \
    --ignore-platform-reqs \
    --optimize-autoloader \
    --apcu-autoloader \
    --no-scripts \
    --no-cache

# ==================================
# STAGE 2: Frontend Assets
# ==================================
FROM node:18-slim AS frontend

WORKDIR /app

# Install build dependencies
RUN apt-get update && apt-get install -y \
    python3 \
    make \
    g++ \
    && rm -rf /var/lib/apt/lists/*

# Copy package files
COPY package*.json ./
RUN npm ci --only=production --silent

# Copy source and build
COPY . .
RUN npm run build 2>/dev/null || echo "No build script found"

# ==================================
# STAGE 3: Production PHP-FPM
# ==================================
FROM php:8.3-fpm-slim AS production

# Set working directory
WORKDIR /var/www/html

# Install system dependencies
RUN apt-get update && apt-get install -y \
    libfreetype6-dev \
    libjpeg62-turbo-dev \
    libpng-dev \
    libzip-dev \
    libicu-dev \
    libonig-dev \
    libxml2-dev \
    libpq-dev \
    mysql-client \
    curl \
    ca-certificates \
    supervisor \
    cron \
    && docker-php-ext-configure gd --with-freetype --with-jpeg \
    && docker-php-ext-install -j$(nproc) \
        pdo_mysql \
        pdo_pgsql \
        mbstring \
        zip \
        gd \
        intl \
        opcache \
        bcmath \
        xml \
        soap \
        exif \
        pcntl \
    && pecl install redis apcu \
    && docker-php-ext-enable redis apcu \
    && apt-get clean \
    && rm -rf /var/lib/apt/lists/* /tmp/* /var/tmp/* /var/cache/apk/*

# Copy PHP configuration
COPY docker/php/php.ini /usr/local/etc/php/conf.d/app.ini
COPY docker/php/php-fpm.conf /usr/local/etc/php-fpm.d/www.conf

# Create application user
RUN groupadd -g 1000 www && \
    useradd -u 1000 -ms /bin/bash -g www www

# Copy vendor dependencies from composer stage
COPY --from=vendor --chown=www:www /app/vendor ./vendor

# Copy built assets from frontend stage  
COPY --from=frontend --chown=www:www /app/public/build ./public/build

# Copy application files
COPY --chown=www:www . .

# Set proper permissions
RUN mkdir -p \
        storage/logs \
        storage/framework/cache \
        storage/framework/sessions \
        storage/framework/views \
        bootstrap/cache \
    && chown -R www:www \
        storage \
        bootstrap/cache \
    && chmod -R 775 \
        storage \
        bootstrap/cache

# Generate optimized autoloader
RUN composer dump-autoload --optimize --apcu --no-dev

# Laravel optimizations (as www user)
USER www
RUN php artisan config:cache 2>/dev/null || true && \
    php artisan route:cache 2>/dev/null || true && \
    php artisan view:cache 2>/dev/null || true

# Copy entrypoint script
USER root
COPY docker/scripts/entrypoint.sh /usr/local/bin/
RUN chmod +x /usr/local/bin/entrypoint.sh

# Health check
HEALTHCHECK --interval=30s --timeout=10s --start-period=60s --retries=3 \
    CMD php artisan tinker --execute="echo 'OK';" || exit 1

# Switch to non-root user
USER www

EXPOSE 9000

ENTRYPOINT ["/usr/local/bin/entrypoint.sh"]
CMD ["php-fpm"]
```

### docker/nginx/default.conf
```nginx
server {
    listen 80;
    server_name _;
    root /var/www/html/public;
    index index.php index.html;

    # Security
    location ~ /\. {
        deny all;
        return 404;
    }

    location ~* \.(env|git|svn|htaccess|htpasswd)$ {
        deny all;
        return 404;
    }

    # Laravel application
    location / {
        try_files $uri $uri/ /index.php?$query_string;
    }

    # PHP-FPM processing
    location ~ \.php$ {
        try_files $uri =404;
        fastcgi_split_path_info ^(.+\.php)(/.+)$;
        fastcgi_pass php:9000;
        fastcgi_index index.php;
        fastcgi_param SCRIPT_FILENAME $realpath_root$fastcgi_script_name;
        include fastcgi_params;
        
        # Security and performance
        fastcgi_hide_header X-Powered-By;
        fastcgi_read_timeout 300s;
        fastcgi_buffers 16 16k;
        fastcgi_buffer_size 32k;
        fastcgi_param HTTPS $https if_not_empty;
    }

    # Static file caching
    location ~* \.(jpg|jpeg|gif|png|css|js|ico|webp|tiff|ttf|svg|woff|woff2|eot)$ {
        expires 1y;
        add_header Cache-Control "public, no-transform, immutable";
        access_log off;
        log_not_found off;
    }

    # API rate limiting
    location ^~ /api/ {
        limit_req zone=api burst=20 nodelay;
        try_files $uri $uri/ /index.php?$query_string;
    }

    # Login rate limiting
    location ^~ /login {
        limit_req zone=login burst=5 nodelay;
        try_files $uri $uri/ /index.php?$query_string;
    }

    # Health check
    location = /health {
        access_log off;
        return 200 "healthy\n";
        add_header Content-Type text/plain;
    }

    # Block exploit attempts
    location ~* (wp-admin|wp-login|xmlrpc) {
        deny all;
        return 404;
    }
}
```

### docker/scripts/entrypoint.sh
```bash
#!/bin/bash
set -e

# Function to wait for database
wait_for_db() {
    echo "Waiting for database connection..."
    while ! mysqladmin ping -h"$DB_HOST" -P"$DB_PORT" -u"$DB_USERNAME" -p"$DB_PASSWORD" --silent; do
        sleep 1
    done
    echo "Database is ready!"
}

# Function to wait for Redis
wait_for_redis() {
    echo "Waiting for Redis connection..."
    while ! redis-cli -h "$REDIS_HOST" -p "$REDIS_PORT" ping > /dev/null 2>&1; do
        sleep 1
    done
    echo "Redis is ready!"
}

# Wait for services if needed
if [ "${WAIT_FOR_DB:-true}" = "true" ]; then
    wait_for_db
fi

if [ "${WAIT_FOR_REDIS:-true}" = "true" ]; then
    wait_for_redis
fi

# Run Laravel setup commands
if [ "${RUN_MIGRATIONS:-false}" = "true" ]; then
    echo "Running database migrations..."
    php artisan migrate --force
fi

if [ "${SEED_DATABASE:-false}" = "true" ]; then
    echo "Seeding database..."
    php artisan db:seed --force
fi

# Create storage link if needed
if [ ! -L /var/www/html/public/storage ]; then
    php artisan storage:link
fi

# Clear and cache Laravel configurations
php artisan config:cache
php artisan route:cache  
php artisan view:cache

echo "Laravel application is ready!"

# Execute the main command
exec "$@"
```

### docker/scripts/wait-for-it.sh
```bash
#!/usr/bin/env bash
# wait-for-it.sh - Wait for a service to be available

WAITFORIT_cmdname=${0##*/}

usage()
{
    cat << USAGE >&2
Usage:
    $WAITFORIT_cmdname host:port [-s] [-t timeout] [-- command args]
    -h HOST | --host=HOST       Host or IP under test
    -p PORT | --port=PORT       TCP port under test
                                Alternatively, you specify the host and port as host:port
    -s | --strict               Only execute subcommand if the test succeeds
    -q | --quiet                Don't output any status messages
    -t TIMEOUT | --timeout=TIMEOUT
                                Timeout in seconds, zero for no timeout
    -- COMMAND ARGS             Execute command with args after the test finishes
USAGE
    exit 1
}

wait_for()
{
    if [[ $WAITFORIT_TIMEOUT -gt 0 ]]; then
        echoerr "$WAITFORIT_cmdname: waiting $WAITFORIT_TIMEOUT seconds for $WAITFORIT_HOST:$WAITFORIT_PORT"
    else
        echoerr "$WAITFORIT_cmdname: waiting for $WAITFORIT_HOST:$WAITFORIT_PORT without a timeout"
    fi
    WAITFORIT_start_ts=$(date +%s)
    while :
    do
        if [[ $WAITFORIT_ISBUSY -eq 1 ]]; then
            nc -z $WAITFORIT_HOST $WAITFORIT_PORT
            WAITFORIT_result=$?
        else
            (echo > /dev/tcp/$WAITFORIT_HOST/$WAITFORIT_PORT) >/dev/null 2>&1
            WAITFORIT_result=$?
        fi
        if [[ $WAITFORIT_result -eq 0 ]]; then
            WAITFORIT_end_ts=$(date +%s)
            echoerr "$WAITFORIT_cmdname: $WAITFORIT_HOST:$WAITFORIT_PORT is available after $((WAITFORIT_end_ts - WAITFORIT_start_ts)) seconds"
            break
        fi
        sleep 1
    done
    return $WAITFORIT_result
}

wait_for_wrapper()
{
    # In order to support SIGINT during timeout: http://unix.stackexchange.com/a/57692
    if [[ $WAITFORIT_QUIET -eq 1 ]]; then
        timeout $WAITFORIT_BUSYTIMEFLAG $WAITFORIT_TIMEOUT $0 --quiet --child --host=$WAITFORIT_HOST --port=$WAITFORIT_PORT --timeout=$WAITFORIT_TIMEOUT &
    else
        timeout $WAITFORIT_BUSYTIMEFLAG $WAITFORIT_TIMEOUT $0 --child --host=$WAITFORIT_HOST --port=$WAITFORIT_PORT --timeout=$WAITFORIT_TIMEOUT &
    fi
    WAITFORIT_PID=$!
    trap "kill -INT -$WAITFORIT_PID" INT
    wait $WAITFORIT_PID
    WAITFORIT_RESULT=$?
    if [[ $WAITFORIT_RESULT -ne 0 ]]; then
        echoerr "$WAITFORIT_cmdname: timeout occurred after waiting $WAITFORIT_TIMEOUT seconds for $WAITFORIT_HOST:$WAITFORIT_PORT"
    fi
    return $WAITFORIT_RESULT
}

echoerr() {
    if [[ $WAITFORIT_QUIET -ne 1 ]]; then echo "$@" 1>&2; fi
}

# process arguments
while [[ $# -gt 0 ]]
do
    case "$1" in
        *:* )
        WAITFORIT_hostport=(${1//:/ })
        WAITFORIT_HOST=${WAITFORIT_hostport[0]}
        WAITFORIT_PORT=${WAITFORIT_hostport[1]}
        shift 1
        ;;
        --child)
        WAITFORIT_CHILD=1
        shift 1
        ;;
        -q | --quiet)
        WAITFORIT_QUIET=1
        shift 1
        ;;
        -s | --strict)
        WAITFORIT_STRICT=1
        shift 1
        ;;
        -h)
        WAITFORIT_HOST="$2"
        if [[ $WAITFORIT_HOST == "" ]]; then break; fi
        shift 2
        ;;
        --host=*)
        WAITFORIT_HOST="${1#*=}"
        shift 1
        ;;
        -p)
        WAITFORIT_PORT="$2"
        if [[ $WAITFORIT_PORT == "" ]]; then break; fi
        shift 2
        ;;
        --port=*)
        WAITFORIT_PORT="${1#*=}"
        shift 1
        ;;
        -t)
        WAITFORIT_TIMEOUT="$2"
        if [[ $WAITFORIT_TIMEOUT == "" ]]; then break; fi
        shift 2
        ;;
        --timeout=*)
        WAITFORIT_TIMEOUT="${1#*=}"
        shift 1
        ;;
        --)
        shift
        WAITFORIT_CLI=("$@")
        break
        ;;
        --help)
        usage
        ;;
        *)
        echoerr "Unknown argument: $1"
        usage
        ;;
    esac
done

if [[ "$WAITFORIT_HOST" == "" || "$WAITFORIT_PORT" == "" ]]; then
    echoerr "Error: you need to provide a host and port to test."
    usage
fi

WAITFORIT_TIMEOUT=${WAITFORIT_TIMEOUT:-15}
WAITFORIT_STRICT=${WAITFORIT_STRICT:-0}
WAITFORIT_CHILD=${WAITFORIT_CHILD:-0}
WAITFORIT_QUIET=${WAITFORIT_QUIET:-0}

# check to see if timeout is from busybox?
WAITFORIT_TIMEOUT_PATH=$(type -p timeout)
WAITFORIT_TIMEOUT_PATH=$(realpath $WAITFORIT_TIMEOUT_PATH 2>/dev/null || echo $WAITFORIT_TIMEOUT_PATH)
if [[ $WAITFORIT_TIMEOUT_PATH =~ "busybox" ]]; then
        WAITFORIT_ISBUSY=1
        WAITFORIT_BUSYTIMEFLAG="-t"

else
        WAITFORIT_ISBUSY=0
        WAITFORIT_BUSYTIMEFLAG=""
fi

if [[ $WAITFORIT_CHILD -gt 0 ]]; then
    wait_for
    WAITFORIT_RESULT=$?
    exit $WAITFORIT_RESULT
else
    if [[ $WAITFORIT_TIMEOUT -gt 0 ]]; then
        wait_for_wrapper
        WAITFORIT_RESULT=$?
    else
        wait_for
        WAITFORIT_RESULT=$?
    fi
fi

if [[ $WAITFORIT_CLI != "" ]]; then
    if [[ $WAITFORIT_RESULT -ne 0 && $WAITFORIT_STRICT -eq 1 ]]; then
        echoerr "$WAITFORIT_cmdname: strict mode, refusing to execute subprocess"
        exit $WAITFORIT_RESULT
    fi
    exec "${WAITFORIT_CLI[@]}"
else
    exit $WAITFORIT_RESULT
fi
```

### build.sh
```bash
#!/bin/bash
set -euo pipefail

# Configuration
PHP_IMAGE_NAME="yourusername/laravel_instasure"
NGINX_IMAGE_NAME="yourusername/laravel_instasure_nginx"
VERSION="v1.0.1"

# Colors
RED='\033[0;31m'
GREEN='\033[0;32m'
YELLOW='\033[1;33m'
BLUE='\033[0;34m'
NC='\033[0m'

log_info() { echo -e "${BLUE}[INFO]${NC} $1"; }
log_success() { echo -e "${GREEN}[SUCCESS]${NC} $1"; }
log_warning() { echo -e "${YELLOW}[WARNING]${NC} $1"; }
log_error() { echo -e "${RED}[ERROR]${NC} $1"; }

# Build PHP container
build_php() {
    log_info "Building PHP container..."
    export DOCKER_BUILDKIT=1
    
    docker build \
        --file docker/php/Dockerfile \
        --target production \
        --tag "$PHP_IMAGE_NAME:$VERSION" \
        --tag "$PHP_IMAGE_NAME:latest" \
        --build-arg BUILD_DATE="$(date -u +'%Y-%m-%dT%H:%M:%SZ')" \
        --build-arg VCS_REF="$(git rev-parse --short HEAD)" \
        --build-arg VERSION="$VERSION" \
        .
    
    log_success "PHP container built: $PHP_IMAGE_NAME:$VERSION"
}

# Build Nginx container
build_nginx() {
    log_info "Building Nginx container..."
    
    docker build \
        --file docker/nginx/Dockerfile \
        --tag "$NGINX_IMAGE_NAME:$VERSION" \
        --tag "$NGINX_IMAGE_NAME:latest" \
        --build-arg BUILD_DATE="$(date -u +'%Y-%m-%dT%H:%M:%SZ')" \
        --build-arg VCS_REF="$(git rev-parse --short HEAD)" \
        --build-arg VERSION="$VERSION" \
        .
    
    log_success "Nginx container built: $NGINX_IMAGE_NAME:$VERSION"
}

# Test containers
test_containers() {
    log_info "Testing containers with docker-compose..."
    
    # Start test environment
    docker-compose -f docker-compose.test.yml up -d
    
    # Wait for services
    sleep 30
    
    # Run basic tests
    docker-compose -f docker-compose.test.yml exec -T php-test php artisan --version
    
    # Cleanup test environment
    docker-compose -f docker-compose.test.yml down -v
    
    log_success "Container tests passed"
}

# Show image information
show_info() {
    log_info "Container Images Information:"
    echo "PHP Container:"
    docker images "$PHP_IMAGE_NAME:$VERSION" --format "table {{.Repository}}\t{{.Tag}}\t{{.Size}}\t{{.CreatedAt}}"
    echo "Nginx Container:"
    docker images "$NGINX_IMAGE_NAME:$VERSION" --format "table {{.Repository}}\t{{.Tag}}\t{{.Size}}\t{{.CreatedAt}}"
}

# Push containers
push_containers() {
    if [ "${SKIP_PUSH:-false}" = "true" ]; then
        log_warning "Skipping push to registry"
        return
    fi
    
    read -p "Push to Docker Hub? (y/N): " -n 1 -r
    echo
    
    if [[ $REPLY =~ ^[Yy]$ ]]; then
        log_info "Pushing to Docker Hub..."
        
        # Push PHP container
        docker push "$PHP_IMAGE_NAME:$VERSION"
        docker push "$PHP_IMAGE_NAME:latest"
        
        # Push Nginx container  
        docker push "$NGINX_IMAGE_NAME:$VERSION"
        docker push "$NGINX_IMAGE_NAME:latest"
        
        log_success "All containers pushed to Docker Hub"
    fi
}

# Main execution
main() {
    log_info "🚀 Building Multi-Container Laravel Application..."
    
    build_php
    build_nginx
    test_containers
    show_info
    push_containers
    
    log_success "🎉 Multi-container build completed!"
    log_info "PHP Image: $PHP_IMAGE_NAME:$VERSION"
    log_info "Nginx Image: $NGINX_IMAGE_NAME:$VERSION"
}

# Handle arguments
while [[ $# -gt 0 ]]; do
    case $1 in
        --no-push) export SKIP_PUSH=true; shift ;;
        --version) VERSION="$2"; shift 2 ;;
        --help)
            echo "Usage: $0 [OPTIONS]"
            echo "  --no-push     Skip pushing to registry"
            echo "  --version     Specify version tag"
            echo "  --help        Show help"
            exit 0 ;;
        *) log_error "Unknown option: $1"; exit 1 ;;
    esac
done

main "$@"
```

### deploy.sh
```bash
#!/bin/bash
set -euo pipefail

# Configuration
APP_DIR="/var/www/laravel-app"
COMPOSE_FILE="docker-compose.prod.yml"
BACKUP_DIR="$APP_DIR/backups"
PHP_IMAGE="yourusername/laravel_instasure"
NGINX_IMAGE="yourusername/laravel_instasure_nginx"
VERSION="v1.0.1"

# Colors
RED='\033[0;31m'
GREEN='\033[0;32m'
YELLOW='\033[1;33m'
BLUE='\033[0;34m'
NC='\033[0m'

log_info() { echo -e "${BLUE}[INFO]${NC} $1"; }
log_success() { echo -e "${GREEN}[SUCCESS]${NC} $1"; }
log_warning() { echo -e "${YELLOW}[WARNING]${NC} $1"; }
log_error() { echo -e "${RED}[ERROR]${NC} $1"; }

# Create backup
create_backup() {
    log_info "Creating backup..."
    mkdir -p "$BACKUP_DIR"
    
    BACKUP_FILE="$BACKUP_DIR/database-$(date +%Y%m%d-%H%M%S).sql"
    
    if docker-compose -f "$APP_DIR/$COMPOSE_FILE" exec -T mysql mysqladmin ping -h localhost --silent 2>/dev/null; then
        docker-compose -f "$APP_DIR/$COMPOSE_FILE" exec -T mysql mysqldump \
            -u root -p"$DB_ROOT_PASSWORD" "$DB_DATABASE" > "$BACKUP_FILE"
        log_success "Database backup created: $BACKUP_FILE"
    else
        log_warning "MySQL not running, skipping backup"
    fi
    
    # Keep only last 5 backups
    ls -t "$BACKUP_DIR"/database-*.sql 2>/dev/null | tail -n +6 | xargs -r rm --
}

# Pull images
pull_images() {
    log_info "Pulling latest images..."
    
    export IMAGE_TAG="$VERSION"
    
    docker pull "$PHP_IMAGE:$VERSION"
    docker pull "$NGINX_IMAGE:$VERSION"
    
    log_success "Images pulled successfully"
}

# Deploy application
deploy_app() {
    log_info "Deploying application..."
    
    cd "$APP_DIR"
    
    # Load environment
    set -a
    source .env
    set +a
    
    # Export image tag
    export IMAGE_TAG="$VERSION"
    export RUN_MIGRATIONS="${RUN_MIGRATIONS:-false}"
    
    # Zero-downtime deployment
    log_info "Stopping old containers..."
    docker-compose -f "$COMPOSE_FILE" down --timeout 30
    
    log_info "Starting new containers..."
    docker-compose -f "$COMPOSE_FILE" up -d --remove-orphans --force-recreate
    
    log_success "Containers started"
}

# Health check
health_check() {
    log_info "Performing health checks..."
    
    # Wait for MySQL
    for i in {1..30}; do
        if docker-compose -f "$APP_DIR/$COMPOSE_FILE" exec -T mysql mysqladmin ping -h localhost --silent 2>/dev/null; then
            log_success "MySQL is ready"
            break
        fi
        [ $i -eq 30 ] && { log_error "MySQL failed to start"; exit 1; }
        sleep 2
    done
    
    # Wait for application
    for i in {1..20}; do
        if curl -f http://localhost/health &>/dev/null; then
            log_success "Application is ready"
            break
        fi
        [ $i -eq 20 ] && { log_error "Application failed to start"; exit 1; }
        sleep 5
    done
}

# Post-deployment
post_deploy() {
    log_info "Running post-deployment tasks..."
    
    cd "$APP_DIR"
    
    # Optimize Laravel
    docker-compose -f "$COMPOSE_FILE" exec -T php php artisan optimize
    
    log_success "Post-deployment completed"
}

# Show status
show_status() {
    log_info "=== Container Status ==="
    cd "$APP_DIR"
    docker-compose -f "$COMPOSE_FILE" ps
    
    echo -e "\n=== Resource Usage ==="
    docker stats --no-stream $(docker-compose -f "$COMPOSE_FILE" ps -q) 2>/dev/null || true
    
    echo -e "\n=== Recent Logs ==="
    docker-compose -f "$COMPOSE_FILE" logs --tail=10
}

# Cleanup
cleanup() {
    log_info "Cleaning up old images..."
    docker image prune -f --filter "until=48h"
    docker container prune -f
    log_success "Cleanup completed"
}

# Main deployment
deploy() {
    log_info "🚀 Starting Multi-Container Laravel Deployment..."
    
    create_backup
    pull_images
    deploy_app
    health_check
    post_deploy
    cleanup
    
    log_success "🎉 Multi-container deployment completed!"
    show_status
}

# Handle commands
case "${1:-deploy}" in
    deploy) deploy ;;
    status) show_status ;;
    backup) create_backup ;;
    cleanup) cleanup ;;
    health) health_check ;;
    help|--help|-h)
        echo "Usage: $0 [COMMAND]"
        echo "Commands:"
        echo "  deploy     Deploy the application (default)"
        echo "  status     Show application status"
        echo "  backup     Create database backup"
        echo "  cleanup    Clean up old images"
        echo "  health     Run health check"
        echo "  help       Show this help"
        ;;
    *) log_error "Unknown command: $1"; exit 1 ;;
esac
```

## 8. Key Benefits of Multi-Container Approach

### ✅ **Advantages:**

| Benefit | Description |
|---------|-------------|
| **🔧 Separation of Concerns** | Each container has one responsibility |
| **📈 Independent Scaling** | Scale PHP workers separately from web server |
| **🛠️ Easier Maintenance** | Update services independently |
| **📊 Better Monitoring** | Separate logs and metrics per service |
| **🚀 Faster Deployments** | Only rebuild changed containers |
| **💪 Production Ready** | Follows Docker best practices |
| **🔍 Better Debugging** | Isolate issues to specific containers |
| **☁️ Cloud Native** | Perfect for Kubernetes/Docker Swarm |

### 📊 **Container Sizes:**

| Container | Size | Purpose |
|-----------|------|---------|
| **PHP (Laravel)** | ~280MB | Application logic, PHP-FPM |
| **Nginx** | ~50MB | Web server, static files |
| **MySQL** | ~600MB | Database (official image) |
| **Redis** | ~40MB | Cache, sessions, queues |
| **PHPMyAdmin** | ~500MB | Database management |

**Total:** ~1.5GB (vs your original 4.6GB = 67% reduction!)
