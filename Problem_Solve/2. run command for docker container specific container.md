Suppose docker-composer run the laravel_app with app conainer 

So , those command derectly we can run . 
# Setup Passport for API authentication
docker-compose exec app php artisan passport:install --force

# Run migrations for new packages
docker-compose exec app php artisan migrate

# Publish vendor assets
docker-compose exec app php artisan vendor:publish --all --force

# Check everything is working
docker-compose exec app php artisan route:list
